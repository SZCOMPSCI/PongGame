# Python Pong

A classic Pong game built with Pythonâ€™s Turtle module. Two players control paddles to hit the ball back and forth while keeping score. The first player to score wins.  

## Features

- Two-player gameplay:
  - Player 1: `W` (up), `S` (down)  
  - Player 2: `Arrow Up` (up), `Arrow Down` (down)
- Continuous paddle movement while holding keys
- Ball and paddle implemented using Python classes
- Score tracking displayed at the top of the screen
- Collision detection for paddles and screen borders
